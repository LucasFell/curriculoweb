Projeto desenvolvido para auxiliar os professores e alunos. Criando um sistema, usando Python e Arduino.

No C.E.S.A.R os professores demoravam muito tempo para realizar a chamada e isso demandava bastente tempo das aulas pela grande quantidade de alunos. A principal tarefa da minha equipe foi otimizar o tempo da chamada, deixando as aulas mais dinâmicas e contínuas. Para isso, colhemos depoimentos dos professores e alunos para nos ajudar a encontrar possíveis soluções. Portanto, sintetizamos uma Software para solucionar o problema, desenvolvendo um sistema usando Python e Arduino, em que o aluno e o professor iriam interagir, tendo o professor uma interface em Python que exibe um questionário, que o aluno responderá no controle feito de Arduíno com radiofrequência, assim confirmando sua presença na aula.
